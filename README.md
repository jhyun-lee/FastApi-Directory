# FastApi-Directory

FastApi를 이용한 디렉토리 관리 웹페이지 입니다. 



![image](https://github.com/jhyun-lee/FastApi-Directory/assets/100923108/a910a4c5-fd46-422c-8f51-cdedb1dbfe3a)


***
필요 라이브러리
==
```
pip install fastapi
pip install jinja2
pip install python-multipart
pip install uvicorn
pip install ffmpeg
```


기능
==

- 파일 업로드
- 파일 다운로드
- 파일 삭제
- 이미지 미리보기
- 영상 변환 및 샘플 출력
- 디렉토리 생성
- 디렉토리 압축/해제
